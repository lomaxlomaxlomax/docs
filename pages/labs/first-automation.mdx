# Lab — Build your first automation

<Warning>Use a sandbox tenant for labs.</Warning>

### Goal
Create an automation that listens for a webhook and creates a ticket.

### Steps
1. **Create a Workflow** → start with the webhook trigger.
2. **Add Steps** → parse JSON, enrich with CMDB, create ticket.
3. **Approvals** → add manager approval for high priority.
4. **Test** → send a sample payload.
5. **Publish** → enable and monitor runs.

<Tip>Timebox to 20–30 minutes. Screenshots go here.</Tip>
