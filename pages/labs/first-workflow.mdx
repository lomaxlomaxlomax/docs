# Lab — Build your first workflow

**Goal:** Create a simple workflow with an on‑demand trigger and one action.

## Steps
1. Create a new workflow (on‑demand trigger)
2. Add a basic action (e.g., HTTP call or transform)
3. Test, view run logs, then publish
4. Add an approval, republish

**Timebox:** 20–30 minutes
