# Getting started

<Info>Estimated time: **10 minutes**</Info>

1. Request access to your BlinkOps tenant.
2. Ensure you have credentials for dev systems (Tenable/Prisma/Entra, etc.).
3. Bookmark this Learning Hub.

### Local preview (optional)

If you want to run these docs locally:

```bash
npm i -g mint
mint dev
# -> http://localhost:3000
```
